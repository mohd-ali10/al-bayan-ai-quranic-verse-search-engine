Dataset Overview

This directory contains Quranic text, translations, and tafsir datasets
used in the AL-BAYAN: AI-Based Quranic Verse Search Engine.

Structure
- quran/: Arabic Quran text and structured datasets
- translations/: English and Urdu translations
- tafsir/: Tafsir Ibn Kathir (English & Urdu) datasets
- scripts/: Data preprocessing scripts

Usage
Currently, the system uses Quran text and translation datasets for
keyword-based and semantic search. Tafsir datasets are included for
future enhancements.
